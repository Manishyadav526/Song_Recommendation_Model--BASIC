# 🎧 MusicAI - Song Recommendation System

MusicAI is a Flask-based music recommendation system that suggests similar songs based on user preferences using similarity algorithms and genre-based filtering.

---

## 🚀 Features

- Personalized song recommendations
- Multi-genre support
- Song search functionality
- Artist diversity filtering
- iTunes song redirection
- Responsive user interface

---

## 🛠️ Tech Stack

- Python
- Flask
- Pandas
- Joblib
- HTML
- CSS
- JavaScript

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python app.py
```
